-p,u,b tag-

p Tag : paragraph tag

bold Tag
underline Tag
<p> p 태그는, 하나의 문장을 나타내기 위해 사용하는 태그입니다.
<b> b 태그는, 굵은 글씨를 표현하기 위해 사용합니다.
<u> u 태그는, 밑줄을 긋기 위해 사용합니다.



-img tag-

<img> 태그는 HTML 문서에 이미지를 표시하기 위해 사용합니다.



-input,textarea tag-

input:
textarea:
<input> 태그는 HTML 문서 내에 사용자가 텍스트를 입력할 수 있도록 해줍니다. <textarea> 태그 역시, 사용자가 텍스트를 입력할 수 있도록 해주는 점에서 같지만, 여러 줄 입력이 가능한 폼을 만들어줍니다.



-button tag-

<button> 태그는 HTML 문서에 클릭할 수 있는 버튼을 추가해줍니다..
